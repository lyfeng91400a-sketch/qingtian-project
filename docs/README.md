# ☀️ 晴天项目看板 v2

轻量级项目看板，让团队工作更清爽。

## 快速开始

直接双击 `index.html` 在浏览器中打开即可使用。

> 登录页已预填演示账号，点击 **登录** 即可进入。

## 功能清单

### 核心功能
- **多项目看板** — 侧边栏列出所有项目，点击即切换看板视图
- **任务 CRUD** — 创建、编辑、删除、详情查看，完整交互闭环
- **项目 CRUD** — 创建、编辑、删除项目，含二次确认
- **状态循环切换** — 看板卡片上点击状态标签即可（待办 → 进行中 → 已完成）
- **看板三列布局** — 待办 / 进行中 / 已完成

### 数据能力
- **localStorage 持久化** — 刷新页面不丢失数据
- **批量导入任务** — 通过管道符 `|` 分隔的文本格式批量导入
- **任务搜索** — 按标题、描述、负责人搜索当前项目任务
- **动态仪表盘** — 统计卡片绑定真实数据（活跃项目数、完成/进行中/待办任务）

### 数据预置
- **穗穗念 v1.0（已完成）** — 8 个已完成任务，涵盖注册登录、内容发布、互动、搜索等功能
- **穗穗念 v1.1（规划中）** — 8 个待办任务，包含 AI 推荐、小程序、付费内容等规划
- **小红书博主排期** — 第二周全周排期（7天），从周一护肤routine到周日数据复盘

### 交互体验
- 登录页 → 仪表盘 → 项目总览 → 看板视图，完整导航流程
- 任务详情弹窗支持评论区
- 删除二次确认防误操作
- Toast 通知反馈
- Escape 键关闭弹窗
- 响应式布局适配平板和手机

## 技术栈

纯前端 HTML + CSS + JavaScript，零后端依赖。

- **数据存储**：`localStorage`
- **样式方案**：CSS 自定义属性（变量）+ Flexbox + Grid
- **架构**：数据层 → 渲染层 → 交互层，分层清晰
